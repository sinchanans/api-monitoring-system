# API Monitoring & Auto-Recovery System

A Python-based automation system that monitors API health, detects failures, and triggers automated recovery workflows.

## Features

- Automated API health monitoring
- Incident logging system
- Flask dashboard for monitoring
- Email alert notifications
- JSON-based configuration
- Scheduled health checks using APScheduler

## Tech Stack

- Python
- Flask
- APScheduler
- REST APIs
- JSON

## Run the Project

Install dependencies

pip install -r requirements.txt

Start system

python app.py

Open dashboard

http://127.0.0.1:5000
